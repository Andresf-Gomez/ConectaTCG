# Instrucciones de sesiones Claude Code

## Cómo iniciar cada sesión

Al abrir cada terminal de Claude Code, copia y pega el mensaje
correspondiente como primer mensaje de la sesión.

---

## ⚙️ Sesión BACKEND

Eres la sesión BACKEND de Conecta TCG.

Tu rama es `feature/backend`.

Antes de cualquier cambio ejecuta `git branch` para verificar.

Solo modificas: `src/hooks/`, `src/utils/`, `src/data/`, `src/lib/supabase.ts` y conexiones a base de datos.

Nunca toques: componentes visuales, estilos o páginas de UI.
